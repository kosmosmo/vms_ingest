# VMS Ingest

Airtable implementation for VMS ingesting.

## Installation

follow the install instructions here. [Mask R-CNN](https://github.com/matterport/Mask_RCNN) and [tf-pose-estimation](https://github.com/ildoonet/tf-pose-estimation)

### Dependencies

- python3.7
- request

### Package Install
```bash
pip install request
```
